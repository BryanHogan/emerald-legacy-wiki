---
title: Stat Changes 
description: Stats changes of Pokemon in Pokemon Emerald Legacy.
---

Stat changes are something that we resort to only if improvements to a pokemon's learnset can’t fix its potential to be useful in a playthrough. We then take a look at what role it is trying to fill or what role it could fill and give it changes to better fit that role. We aren’t trying to make every pokemon an S tier pokemon because then we are back to the same point where we started off with everything being equal filling up the same roles. If everything's the same then nothing feels special.
